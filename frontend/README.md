# Frontend – NovaTech-Onboarding-Assistent

## Beschreibung

Das Frontend wurde mit React, TypeScript und Vite entwickelt. Es stellt die Benutzeroberfläche des NovaTech-Onboarding-Assistenten bereit und kommuniziert über eine REST-API mit dem FastAPI-Backend.

## Verwendete Technologien

- React
- TypeScript
- Vite
- EventSource (Server-Sent Events)
- CSS

## Funktionen

- Chatoberfläche
- Automatische Begrüßungsnachricht
- Conversation Memory
- Streaming der KI-Antworten
- Aktivitätsverlauf
- Fortschrittsanzeige
- Kontextmenü
- Fehlerbehandlung
- Responsive Chatoberfläche

## Projekt starten

```bash
npm install
npm run dev
```

Das Frontend ist anschließend unter

```
http://localhost:5173
```

erreichbar.

Das Backend muss parallel gestartet werden.
